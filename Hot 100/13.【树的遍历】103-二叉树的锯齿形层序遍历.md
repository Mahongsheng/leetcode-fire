# 【树的遍历】103-二叉树的锯齿形层序遍历

妥善利用好栈即可，这个代码写的不好，应该直接层序遍历，然后奇数层正常`add`，偶数层用`add(int index,Object o)`

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public List<List<Integer>> zigzagLevelOrder(TreeNode root) {
        List<List<Integer>> ans = new ArrayList<>();
        if (root == null) {
            return ans;
        }
        Stack<TreeNode> preStack = new Stack<>(), nextStack = new Stack<>();
        preStack.push(root);
        boolean ifOddLayer = true;
        while (!preStack.isEmpty() || !nextStack.isEmpty()) {
            List<Integer> currAns = new ArrayList<>();
            if (ifOddLayer) {
                // 奇数层
                while (!preStack.isEmpty()) {
                    TreeNode curr = preStack.pop();
                    currAns.add(curr.val);
                    if (curr.left != null) {
                        nextStack.push(curr.left);
                    }
                    if (curr.right != null) {
                        nextStack.push(curr.right);
                    }
                }
            } else {
                // 偶数层
                while (!preStack.isEmpty()) {
                    TreeNode curr = preStack.pop();
                    currAns.add(curr.val);
                    if (curr.right != null) {
                        nextStack.push(curr.right);
                    }
                    if (curr.left != null) {
                        nextStack.push(curr.left);
                    }
                }
            }
            ifOddLayer = !ifOddLayer;
            preStack = nextStack;
            nextStack = new Stack<>();
            ans.add(currAns);
        }
        return ans;
    }
}
```
